A collection of small **data visualization exercises** created while learning scientific computing with **Python, NumPy, and Matplotlib**.

These notebooks demonstrate how to generate data, analyze functions, and visualize mathematical relationships.

---

# 📂 Included Examples

### 1️⃣ Sine Function with Extrema
`sin_plot.ipynb`

Visualization of the function **sin(x)** on the interval `[-4π, 4π]` with highlighted:

- maxima
- minima
- grid and labeled axes

---

### 2️⃣ Polynomial Functions
`polynomial_functions.ipynb`

Comparison of several functions:

- f(x) = x  
- f(x) = x² + 1  
- f(x) = x³ + 1  

Demonstrates plotting multiple functions on one chart.

---

### 3️⃣ Parametric Curves
`parametric_curve.ipynb`

Visualization of parametric equations:

φ(t) = [a cos(t), a sin(t)]

for different values of **a**, showing how parameters affect the shape of the curve.

---

### 4️⃣ Plotting Data from File
`plot_from_file.ipynb`

Reads numerical data from a **text file** and visualizes it using Matplotlib.

Demonstrates:

- file handling
- data parsing
- plotting real datasets
