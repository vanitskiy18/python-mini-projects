# ❌ Tic Tac Toe (PyQt)

![Python](https://img.shields.io/badge/Python-3.x-blue)
![PyQt](https://img.shields.io/badge/GUI-PyQt-orange)
![Game](https://img.shields.io/badge/Project-Mini%20Game-lightgrey)

A simple graphical implementation of the classic **Tic Tac Toe** game written in Python using **PyQt**.

The project was created as a small exercise to practice GUI programming and basic game logic.

---

## 🎮 Features

- Graphical interface built with **PyQt**
- Classic **3×3 Tic Tac Toe gameplay**
- Two-player game on a single computer
- Basic win condition detection
- Simple and clean project structure

## 🚀 Running the Game

1. Make sure Python is installed.
2. Install PyQt if it is not installed:

```bash
pip install PyQt6
```
3. Run the game:
python tic-tac-toe.py
The game window will open and players can start playing immediately.
