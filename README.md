Some of them were created as learning exercises, others as small standalone applications.

# 📂 Project Structure

## 🖥️ Tkinter Applications

Small GUI applications built using **Tkinter**.

Projects include:

- **Memory Translator** – a small vocabulary learning game
- **Quadratic Equation Solver** – solves quadratic equations using a graphical interface
- **Sticks Game** – a simple strategy game against the computer

---

## 🧩 PyQt Applications

GUI programs created with the **PyQt framework**.

Projects include:

- **Virtual Keyboard**
- **Tic Tac Toe**
- **Tax Calculator** *(separate repository)*

---

## 🔐 Cryptography

Simple implementations of classical encryption algorithms with a graphical interface.

Includes:

- Caesar cipher
- Atbash cipher
- encryption tools
- decryption tools

---

## 🧠 Algorithms

Implementations of algorithms studied during algorithmics practice.

Examples include:

- Quick Sort
- Merge Sort
- Inversion counting
- Greedy coin algorithm
- Street lights coverage problem
- Heap sort with transaction sorting

---

## 🎮 Console Games

Small **text-based games** written for learning purposes.

Includes:

- **Imposter Game** – role-based console game using OOP
- **Russian Roulette** – a multiplayer elimination game

---

### 📊 Matplotlib Visualizations

Examples of mathematical and data visualizations created using **NumPy** and **Matplotlib**.

These notebooks demonstrate how to generate numerical data and visualize mathematical relationships.

Projects include:

- **Sine Function Plot** – visualization of sin(x) on the interval [-4π, 4π] with highlighted maxima and minima
- **Polynomial Functions Plot** – comparison of functions such as x, x² + 1, and x³ + 1
- **Parametric Curve Visualization** – plotting parametric equations for different parameter values
- **Plot From File** – reading numerical data from a text file and visualizing it with Matplotlib
