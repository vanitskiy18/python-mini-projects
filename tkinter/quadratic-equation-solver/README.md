# 🧮 Quadratic Equation Solver

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)
![Math](https://img.shields.io/badge/Project-Math%20Tool-lightgrey)

A simple graphical application for solving **quadratic equations** written in Python using **Tkinter**.

The program allows users to input the coefficients of a quadratic equation and instantly compute the solutions.

---

## ✨ Features

- Solve equations of the form:
  - ax² + bx + c = 0
- Automatic calculation of the **discriminant**
- Displays:
  - two real solutions
  - one solution
  - or informs when no real solutions exist
- Simple graphical interface built with **Tkinter**
