# 🧠 Memory Translator Game

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)
![Game](https://img.shields.io/badge/Project-Learning%20Game-lightgrey)

A small educational **word memorization game** written in Python using **Tkinter**.

The game helps users practice vocabulary by selecting the correct translation of a given word using an on-screen keyboard.

---

## 🎮 Game Idea

The program displays a word and the player must **type the correct translation** using the graphical keyboard.

The goal of the game is to help with **memorizing vocabulary through repetition and interaction**.

---

## ✨ Features

- Interactive **on-screen keyboard**
- Vocabulary practice through translation
- Simple and lightweight graphical interface
- Word list stored in a separate dictionary file
- Immediate feedback for user answers

---

tkinter-memory-translator

## 🚀 Running the Game

1. Make sure Python is installed.

2. Run the game:

```bash
python memory_game.py
```

3. Use the graphical keyboard to type the translation of the displayed word.
