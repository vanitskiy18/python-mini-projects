# 🔐 Classical Cipher Tool

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)
![Cryptography](https://img.shields.io/badge/Cryptography-Classical-lightgrey)

A small graphical tool for experimenting with **classical encryption algorithms**.

The application allows users to encrypt and decrypt text using simple historical ciphers.

This project was created as a learning exercise while studying Python, GUI programming and basic cryptography concepts.

---

## ✨ Features

- Encrypt text using **Caesar cipher**
- Encrypt text using **Atbash cipher**
- Decrypt encrypted text
- Simple graphical interface built with **Tkinter**
- Modular code structure

## 🔐 Implemented Ciphers

---

### Caesar Cipher

A classical substitution cipher where each letter in the plaintext is shifted by a fixed number of positions in the alphabet.

Example:

HELLO → KHOOR

---

### Atbash Cipher

A monoalphabetic substitution cipher that maps the alphabet to its reverse.

Example:

A ↔ Z
B ↔ Y
C ↔ X
